English

SHELL is also a programming language:

- The syntax is simple and straightforward
- Most short scripts work right the first time
- Debugging is straightforward

The shell program interprets user commands, which are either directly entered by the user, or which can be read from a file called the shell script or shell program.

Shell scripts are interpreted, not compiled. The shell reads commands from the script line per line and searches for those commands on the system, while a compiler converts a program into machine readable form, an executable file - which may then be used in a shell script.

Español

SHELL también es un lenguaje de programación:

- La sintaxis es simple y directa
- La mayoría de los scripts cortos funcionan bien desde la primera vez
- Depurar es sencillo

El programa shell interpreta comandos del usuario, que pueden ser ingresados directamente o leídos desde un archivo llamado script de shell o programa de shell.

Los scripts de shell son interpretados, no compilados. El shell lee los comandos del script línea por línea y los busca en el sistema, mientras que un compilador convierte un programa en una forma legible por máquina, un archivo ejecutable, que luego puede usarse en un script de shell.
